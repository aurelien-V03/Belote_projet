Project Name : Tux Letter Game

**************************
Objectif du jeu : L'objectif de ce projet à été de développer un jeu en 3D autour de différentes technologies
comme Java (pour l'environnement 3D, les classes métiers etc...), XML (pour stocker certains paramètres du jeu 
et des données de jeu), XSLT...

Durant une partie vous incarnez un personnage (Tux) qui doit aller récuperer les lettres d'un mot sur un plateau,
le t emps est limité et Tux doit récuperer toutes les lettres du mot pour ne pas perdre, par
éxemple un mot peut etre "chat", dans ce cas Tux devra aller récupérer la lettre 'C' puis la lettre 'H' etc...

Il éxiste différents niveau de difficultés (de 1 à 4), en général un niveau dificile aura des mots assez long par éxemple.

Le joueur a la possibilité d'enregistrer son score dans un base de donnée en XML et rejouer plus tard avec le meme "compte"
